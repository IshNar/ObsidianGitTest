## A3. 프로젝트 상세 — 게임 생태계

### A3-1. 계정 거래 사이트 + 게임 시세 트래커

- **수익 모델**: 거래 수수료 3~5% + 시세 트래커 프리미엄 구독 $5/월
- **핵심 기능**: 에스크로 결제, 게임별 계정/아이템 시세 차트, AI 가격 예측 알림
- **차별화**: 주식 앱 개발 역량을 게임 경제에 적용 — 시세 추적/예측은 경쟁자에 없음
- **도구**: Stitch (UI) → Cursor 3 (Design Mode + 병렬 에이전트) → Claude Code (에스크로/결제 로직)
- **우선순위**: ★★★★★ — 가장 먼저 MVP 출시

### A3-2. 홍보 사이트 (게임 커뮤니티/스트리머 특화)

- **수익 모델**: 홍보 서비스 수수료, 스트리머 매칭 수수료
- **핵심 기능**: 게임 길드/클랜 홍보, 스트리머 후원/매칭, 게임 커뮤니티 광고
- **차별화**: 범용 홍보가 아닌 게임 생태계 특화 — 계정 거래 사이트와 같은 고객층
- **도구**: Stitch (UI) → Cursor 3 (Design Mode + 개발) 
- **우선순위**: ★★★★ — 계정 거래 사이트와 동시 또는 직후 런칭

### A3-3. Parallel App (게임 다계정 특화)

- **수익 모델**: 월 구독 $3~5 (APK 직접 배포)
- **핵심 기능**: 하나의 모바일에서 동일 게임 다계정 동시 실행
- **차별화**: 게임 커뮤니티 직접 배포로 앱스토어 정책 회피, 계정 거래 사이트와 시너지

- 적응형 시험 (내 수준에 따라 난이도 변동)

- 새 문제 유형: 단어 완성하기, 일상 읽기, 듣고 응답 고르기 등
- 점수 체계: 1~6 척도 병기 (기존 0~120도 2년간 유지)

## B4. 스피킹 템플릿 & 만점 훈련

### Task 1 (Independent) — 의견 말하기

```
I believe that [의견].
I feel this way for two reasons.

First of all, [이유1].
For example, [구체적 예시].

Secondly, [이유2].
To illustrate, [구체적 예시].

That's why I think [의견 반복].
```

### Task 2 (Integrated) — 읽기+듣기→말하기

```
The reading passage discusses [주제].
According to the reading, [핵심 내용].

However, in the lecture, the professor [agrees/disagrees].
The professor explains that [이유1].
Additionally, the professor mentions [이유2].

So the lecture [supports/challenges] the reading.
```

### Task 3 (Integrated) — 듣기→말하기

```
The lecture is about [주제].
The professor explains [핵심 개념].

First, the professor gives an example of [예시1].
He/she explains that [상세].

Then, the professor describes [예시2].
According to the professor, [상세].

This demonstrates [개념 정리].
```

> **연습법**: 매일 1개 Task를 15초 준비 → 45초 녹음 → 다시 듣고 개선점 메모 → 재녹음
> **만점 추가**: 녹음을 Claude에게 텍스트로 옮겨서 문법/표현 첨삭 → 수정본으로 재녹음

### 스피킹 30점 추가 훈련

1. **원어민 전환 표현 암기** (매일 2개씩)
   - "That being said,..." / "What I mean by that is..."
   - "To put it another way,..." / "Building on that point,..."
   - "This ties back to..." / "The way I see it,..."

2. **발음 교정 루틴** (매일 5분 — 19:30 고정)
   - 문제 발음 집중: th, r/l, v/b 구분
   - 강세/억양: 문장 내 키워드에 자연스러운 강세
   - Shadowing: TED Talk 1분 구간 따라 말하기

3. **유창성 훈련** (수/금 저녁 스피킹 시간에 포함)
   - 랜덤 주제로 60초 즉흥 말하기 (타이머 필수)
   - 멈춤/필러("um","uh") 횟수 세기 → 매주 줄이기

## B5. 라이팅 고득점 전략

### Academic Discussion (10분)

```
I agree/disagree with [의견] for the following reasons.

To begin with, [이유1 + 구체적 예시].
This is important because [근거].

Furthermore, [이유2 + 구체적 예시].
Research has shown that [근거].

In conclusion, [요약].
```

### 고득점 표현 10개 (암기 필수)

1. It is widely acknowledged that...
2. This phenomenon can be attributed to...
3. A compelling example of this is...
4. From my perspective,...
5. This approach has the advantage of...
6. On the other hand, one might argue that...
7. The evidence suggests that...
8. In light of these factors,...
9. This is particularly relevant because...
10. Taking all these points into consideration,...

> **연습법**: 매일 1편 작성 → Claude 첨삭 → 피드백 반영 다시 작성
> **만점 추가**: 같은 주제 3번 쓰기 — 1차(초안) → 2차(피드백 반영) → 3차(만점 완성본) → 암기

### 라이팅 30점 추가 훈련

1. **고급 문장 구조 5개 체화** (매일 1개씩 에세이에 적용)
   - 양보: "While it is true that..., it is important to note that..."
   - 도치: "Not only does this..., but it also..."
   - 분사: "Having considered both perspectives, I believe..."
   - 가정법: "Had the situation been different, the outcome would have..."
   - 강조: "It is precisely this aspect that makes..."

2. **퇴고 체크리스트** (매 에세이 작성 후 2분)
   - [ ] 주제문이 명확한가?
   - [ ] 각 문단에 구체적 예시가 있는가?
   - [ ] 고급 표현이 3개 이상 자연스럽게 쓰였는가?
   - [ ] 문법 오류 (주어-동사 일치, 시제, 관사) 없는가?
   - [ ] 결론이 서론과 일관성 있게 마무리됐는가?
