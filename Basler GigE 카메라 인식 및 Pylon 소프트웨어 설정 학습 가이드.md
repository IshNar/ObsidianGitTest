본 학습 가이드는 Basler pylon 26.01 소프트웨어 설치 후 GigE 카메라(특히 acA1920-25gm 모델)가 인식되지 않는 문제를 해결하기 위한 기술적 절차와 하드웨어 요구사항을 체계적으로 정리한 문서입니다.

--------------------------------------------------------------------------------

1. 이해도 확인 퀴즈 (단답형 10문항)

**질문 1:** pylon 소프트웨어 설치 후 윈도우 네트워크 연결 창에서 카메라가 보이지 않을 때 가장 먼저 확인해야 할 전용 앱은 무엇입니까? **질문 2:** pylon IP Configurator에서 카메라 상태가 'Incomplete'로 표시될 때의 주된 원인과 해결 방법은 무엇입니까? **질문 3:** pylon 설치 시 'Runtime' 옵션만 선택했을 때 발생할 수 있는 드라이버 누락 문제는 무엇입니까? **질문 4:** GigE Vision Filter Driver가 정상적으로 작동하고 있는지 네트워크 어댑터 속성에서 확인하는 방법은 무엇입니까? **질문 5:** 네트워크 보안 소프트웨어가 GigE 카메라 인식을 방해하는 기술적인 이유는 무엇입니까? **질문 6:** 카메라 인식이 불안정할 때 네트워크 어댑터(NIC)의 '점보 프레임(Jumbo Packet)'을 어떤 값으로 설정해야 합니까? **질문 7:** 카메라가 아예 인식되지 않는 단계에서 하드웨어적인 연결 상태를 파악하기 위해 가장 먼저 확인해야 할 부분은 무엇입니까? **질문 8:** PoE(Power over Ethernet) 기능을 지원하지 않는 일반 PC 랜카드에 카메라를 직결했을 때 발생하는 문제는 무엇입니까? **질문 9:** acA1920-25gm 모델의 전원 공급 방식에 대해 설명하고, 이 모델이 랜선만으로 구동 가능한지 답하시오. **질문 10:** 소프트웨어 및 전원 설정이 완벽함에도 LED가 켜지지 않는다면 어떤 물리적 결함을 의심해야 합니까?

--------------------------------------------------------------------------------

2. 퀴즈 정답지

1. **정답:** pylon IP Configurator입니다. 이 앱은 윈도우 기본 네트워크 설정과 별개로 연결된 카메라의 IP 상태를 직관적으로 보여줍니다.

2. **정답:** PC의 이더넷 카드 IP 대역과 카메라의 IP 대역이 서로 맞지 않을 때 발생합니다. 'Auto IP' 또는 'Static IP' 설정을 통해 PC와 동일한 서브넷으로 맞춰주어야 합니다.

3. **정답:** pylon GigE Vision Filter Driver가 누락될 수 있습니다. 이 드라이버가 없으면 윈도우 스택에서 카메라 패킷을 가로채지 못해 인식이 불가능해집니다.

4. **정답:** 제어판의 어댑터 설정 변경에서 해당 랜카드를 우클릭한 후 [속성] 목록에 'pylon GigE Vision Filter Driver'가 체크되어 있는지 확인하면 됩니다.

5. **정답:** IP Configurator는 UDP 브로드캐스트 패킷을 사용하여 카메라를 찾는데, 보안 프로그램이 이 통신을 차단할 경우 카메라 리스트가 비어 있게 됩니다.

6. **정답:** 9014 Bytes 또는 랜카드가 지원하는 최대치로 설정해야 합니다. 이는 데이터 전송 효율을 높이고 인식 안정성을 확보하는 데 필수적입니다.

7. **정답:** 카메라 뒷면의 이더넷 포트 LED 상태를 확인해야 합니다. LED가 꺼져 있다면 물리적 전원 공급에 문제가 있는 것입니다.

8. **정답:** 전원이 공급되지 않아 카메라가 작동하지 않습니다. 일반 랜카드는 전력 전송 기능이 없으므로 PoE 인젝터나 전용 스위칭 허브를 별도로 사용해야 합니다.

9. **정답:** 해당 모델은 별도의 외부 전원(12V~24V DC)이 필요합니다. 모델명의 'gm' 시리즈는 PoE를 지원하지 않으므로 랜선만으로는 전원 공급이 불가능합니다.

10. **정답:** 랜 케이블의 물리적 단선(케이블 불량)을 의심해야 합니다. 검증된 다른 케이블로 교체하여 테스트를 진행해야 합니다.

--------------------------------------------------------------------------------

3. 심화 학습을 위한 에세이 주제

1. **GigE Vision Filter Driver의 역할과 중요성:** 일반적인 네트워크 드라이버와 비교하여 이 필터 드라이버가 고해상도 영상 데이터 전송에서 수행하는 핵심 기능에 대해 논하시오.

2. **산업용 카메라의 전원 공급 체계 비교:** PoE 방식과 외부 전원(Hirose 커넥터) 방식의 장단점을 분석하고, 시스템 구성 환경에 따른 적절한 선택 기준을 제시하시오.

3. **네트워크 어댑터(NIC) 최적화가 시스템 성능에 미치는 영향:** 점보 프레임 설정과 전원 관리 옵션 해제가 왜 산업용 비전 시스템의 안정성에 필수적인지 기술적 근거를 바탕으로 설명하시오.

4. **단계별 트러블슈팅 프로세스 구축:** 카메라 미인식 상황 발생 시, 물리 계층(Physical Layer)부터 소프트웨어 계층까지 확인해야 할 절차를 논리적 순서에 따라 설계하시오.

5. **가상화 환경에서의 GigE 카메라 통신:** VMware 등 가상 머신 사용 시 발생할 수 있는 네트워크 브릿지 설정 이슈와 IP 인식 문제의 상관관계에 대해 서술하시오.

4. 핵심 용어 사전 (Glossary)

| 용어                            | 정의                                                                   |
| ----------------------------- | -------------------------------------------------------------------- |
| **pylon IP Configurator**     | Basler 카메라의 IP 주소를 할당하고 네트워크 상태(Incomplete 등)를 확인/수정하는 전용 유틸리티.      |
| **GigE Vision Filter Driver** | 네트워크 카드와 애플리케이션 사이에서 카메라 패킷을 효율적으로 가로채고 처리하는 전용 드라이버.                |
| **Incomplete 상태**             | 카메라와 PC의 서브넷(Subnet)이 일치하지 않아 통신이 불가능한 상태를 나타내는 pylon의 상태 메시지.       |
| **Jumbo Frame (점보 프레임)**      | 이더넷 프레임 크기를 기본 1,500바이트에서 9,014바이트 등으로 확장하여 오버헤드를 줄이는 설정.            |
| **PoE (Power over Ethernet)** | 별도의 전원선 없이 이더넷 케이블 하나로 데이터 전송과 전력을 동시에 공급하는 기술.                      |
| **UDP Broadcast**             | 네트워크상의 모든 장치에 패킷을 보내 장치를 탐색하는 통신 방식. IP Configurator가 카메라를 찾을 때 사용함. |
| **Hirose 커넥터**                | 산업용 카메라에 전원이나 I/O 신호를 공급하기 위해 사용되는 원형 다핀(6핀/12핀) 커넥터 규격.             |
| **PoE 인젝터 (Injector)**        | PoE를 지원하지 않는 일반 스위치나 랜카드 환경에서 랜선에 전력을 실어주는 중간 장치.                    |
| **Static IP (고정 IP)**         | 장치에 수동으로 직접 부여한 고정된 주소. 통신 안정성을 위해 산업용 환경에서 권장됨.                     |
| **Auto IP (LAA/DHCP)**        | 장치가 자동으로 네트워크 환경에 맞는 IP 주소를 할당받도록 설정하는 방식.                           |
