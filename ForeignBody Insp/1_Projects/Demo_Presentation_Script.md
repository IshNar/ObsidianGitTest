# 🎙️ 바이알 이물 검사 시스템(ForeignBodyInsp) 시연 발표 자료

> **발표 일자**: 2026. 03. 09 (월)  
> **발표자**: 박민우 (작성자/개발자)  
> **시연 목표**: Rule-based(규칙 기반) 영상 처리와 Deep Learning(AI) 기술이 어떻게 융합되어 바이알(Vial) 내부의 미세 이물과 기포를 정확히 분류하고 검출하는지 시연합니다.

---

## 🕒 [00:00 - 01:30] 1. 도입 및 프로그램 구동 (Introduction & Launch)

**[스크립트]**
"안녕하십니까, 이번 프로젝트 시연을 맡은 담당자입니다. 
지금부터 보여드릴 **'ForeignBodyInsp'** 시스템은 의약품 바이알 내부의 이물질을 검출하기 위해 개발된 딥러닝 융합 비전 검사 시스템입니다.
현장에 개발 환경(파이썬 등)이 없어도 작동할 수 있도록 1-Click 실행(Exe) 형태로 개발되었습니다. 바로 프로그램을 실행해 보겠습니다."

**[시연 액션]**
1. 미리 빌드해 둔 `dist/ForeignBodyInsp/ForeignBodyInsp.exe` (혹은 `run.bat`) **더블클릭 실행**
2. 메인 대시보드(GUI) 화면이 뜨는 것 확인
3. 카메라 연결 시연:
   - (실제 Basler 카메라가 있는 경우) 상단 메뉴에서 **[카메라 연결]** 시연
   - (실제 카메라가 없는 경우) 상단 메뉴에서 **[폴더 폴링(가상 카메라)]** 기능을 통해 미리 준비된 샘플 이미지 폴더 로드

---

## 🕒 [01:30 - 03:00] 2. 룰 기반(Rule-based) 전처리 및 한계 시연

**[스크립트]**
"먼저 전통적인 룰 기반(Rule-based) 영상 처리를 활용한 1차 검출 과정을 보여드리겠습니다. 카메라로 이미지가 들어오면 시스템은 실시간으로 영상의 대비를 높이고 이진화를 진행합니다."

**[시연 액션]**
1. 메인 화면에서 영상이 실시간으로 갱신되는 모습(Live View) 시연
2. 상단 메뉴 **[검사 설정] -> [룰 기반 파라미터 셋업]** 다이얼로그 열기
3. 슬라이더바(이진화 임계값, 이물질 크기 기준 등)를 조절하며, **화면의 노이즈가 제거되고 검은 점(이물/기포)만 남는 과정**을 실시간으로 시연
4. **문제 상황(한계점) 연출**: "하지만 보시는 것처럼 룰 기반 방식만으로는 동그란 '기포'와 실제 '이물질'을 형태만으로 완벽하게 구분하기 어렵습니다."

---

## 🕒 [03:00 - 05:00] 3. 딥러닝(AI) 융합 분류 시연 (Core Feature)

**[스크립트]**
"이를 해결하기 위해 저희 시스템은 2차 딥러닝 엔진을 융합했습니다. 룰 기반에서 '의심 객체'로 찾아낸 영역을 절개(Crop)하여, 내장된 PyTorch AI 모델(ResNet/EfficientNet 등)에 넘깁니다. 
AI가 해당 객체의 미세한 질감과 패턴을 분석해 이것이 '정상 기포'인지 '치명적 이물질'인지 최종 판정합니다."

**[시연 액션]**
1. 'Start Inspection(검사 시작)' 버튼 클릭
2. 샘플 이미지가 들어올 때, 화면 우측 또는 하단의 **[결과 로그/판정 뷰]**에 주목시킴
3. 기포 이미지가 들어왔을 때: "Rule 기반에선 검출되었지만, AI가 기포(Bubble)로 판단하여 최종 **[PASS(양품)]** 처리합니다."
4. 실제 이물질 이미지가 들어왔을 때: "AI가 이물질(Foreign Body)로 인식하여 최종 **[FAIL(불량)]** 알람과 함께 빨간색 박스로 마킹합니다."
5. (선택 사항) 객체 탐지(YOLO) 기능이 켜져 있다면, 전체 화면에서 이물이 박스 쳐지는 모습 함께 보여주기

---

## 🕒 [05:00 - 06:30] 4. 실시간 라벨링 및 모델 학습 (UX/UI & Maintenance)

**[스크립트]**
"산업 현장에서는 새로운 유형의 이물이나 약액이 추가될 때마다 AI 모델을 지속적으로 업데이트(튜닝)해야 합니다. 
보통은 개발자에게 이미지를 보내고 기다려야 하지만, 우리 시스템에는 **사용자가 직접 현장에서 라벨링하고 즉시 학습시킬 수 있는 탭**이 내장되어 있습니다."

**[시연 액션]**
1. 우측 상단의 **[Classification 탭]** 또는 **[YOLO Annotation 탭]** 클릭
2. 방금 검사했던 이미지 리스트 중 오판된 이미지(또는 애매한 이미지)를 클릭
3. 버튼 한 번으로 해당 이미지를 '정상(Bubble)' 또는 '불량(Foreign Body)' 폴더로 쉽게 드래그 앤 드롭 / 라벨링하는 과정 시연
4. "이렇게 데이터가 쌓이면 탭 내부의 [학습 시작] 버튼을 눌러, 기존 모델에 새로운 특징을 추가로 파인튜닝(Fine-tuning)할 수 있습니다."
